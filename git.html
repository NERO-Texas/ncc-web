Keyboard Mapping

  • A Selects the "Add Skill" drop-down.
  • 1-9 Adds/removes a spell by level
  • = Toggles Spell Add Mode on/off
  • + Sets Spell Add Mode on
  • - Sets Spell Add Mode off
  • P Toggles the primary school
  • S Toggles the selected school
